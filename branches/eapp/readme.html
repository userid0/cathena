Introduction

Welcome to the world of eAthena!
eAthena is an open source project written to emulate the server of the popular MMORPG Ragnarok Online.
Intro Changelog Features NPC Features Setup GM Commands FAQ Sites

Thank you for downloading eAthena SVN series.

eAthena is an emulator for Ragnarok Online servers. It's written in C, and is cross-platform. The officially supported platforms are Win32 and Linux.

We hope you enjoy using it!

eAthena is licensed under the GPL, meaning it's Open Source.
Anonymous eAthena SVN access is available at http://dev.deltaanime.net:8080/svnroot/eathena, we do not support it though.

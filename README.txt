------------------
- /branches
------------------
Contains all main development branches

* The only time anyone should submit to the stable branch is when they
  have a working and tested bugfix.


------------------
- /trunk
------------------
As of SVN Revision 5091, all untested bugfixes and features go into /trunk.
Once a bugfix or feature has been tested and proven to be working,
it will be added to the stable branch by either Valaris or Wizputer.


------------------
- /devel
------------------
Devs' personal folders for testing new features, etc...
